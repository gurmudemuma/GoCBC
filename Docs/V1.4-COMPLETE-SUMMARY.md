```powershell
# 1. Package chaincode
cd chaincodes\coffee
tar czf coffee_1.4.tar.gz *.go go.mod go.sum

# 2. Install on all 6 peers
peer lifecycle chaincode install coffee_1.4.tar.gz

# 3. Approve for all 6 organizations (sequence 5)
peer lifecycle chaincode approveformyorg ...

# 4. Commit to coffeechannel
peer lifecycle chaincode commit -C coffeechannel -n coffee -v 1.4 --sequence 5

# 5. Test
peer chaincode query -C coffeechannel -n coffee -c '{"Args":["QueryAllExporters"]}'
```

## Key Features Implemented

### SWIFT Integration ✅
- MT103 (Single Customer Credit Transfer)
- MT700 (Documentary Credit/LC)
- SWIFT reference tracking
- Intermediary banks support
- Status tracking: SENT → IN_TRANSIT → RECEIVED → SETTLED

### NBE Retention Policy ✅
- Configurable retention rates (e.g., 40%)
- Automatic calculation:
  - Retained amount: $50,000 × 40% = $20,000 (kept in forex)
  - Converted amount: $50,000 × 60% = $30,000 × 115.5 ETB = 3,465,000 ETB
- Exchange rate application
- Policy history tracking

### Bill of Lading ✅
- B/L number and issue date
- Vessel/truck information
- Departure and destination ports
- ETA vs actual arrival
- GPS/container tracking numbers

### Complete Audit Trail ✅
- All transactions immutable on blockchain
- Timestamp tracking
- Status transition history
- Multi-organization visibility
